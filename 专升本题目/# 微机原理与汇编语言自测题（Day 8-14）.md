### 二、程序填空题
完成以下程序，实现数组中最大值的查找：
```asm
DATA SEGMENT
    ARRAY DB 5, 12, 8, 3, 15, 7
    MAX DB ?
DATA ENDS
CODE SEGMENT
    ASSUME CS:CODE, DS:DATA
START:
    MOV AX, DATA
    MOV DS, AX
    LEA SI, ARRAY
    MOV AL, [SI]        ; 假设第一个为最大
    MOV CX, ____        ; 填空1：循环次数
L1: INC SI
    CMP AL, ____        ; 填空2
    JAE NEXT
    MOV AL, [SI]
NEXT: LOOP L1
    MOV MAX, AL
    MOV AH, 4CH
    INT 21H
CODE ENDS
    END START
```

1. 编写完整程序，实现两个16位数的加法，结果存入内存

3. 编写程序将字符串中的小写字母转换为大写字母
4. 编写程序实现冒泡排序（5个字节数据）

### 答案

二、填空1: 5  填空2: [SI]
