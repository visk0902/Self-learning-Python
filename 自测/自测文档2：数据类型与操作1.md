### 三、简答题
1. 列表是可变的，可以修改元素；元组是不可变的，不能修改元素。列表使用[]定义，元组使用()定义。

2. 字典是键值对数据结构，使用{}定义，通过键访问值；集合是无序、不重复的元素集合，使用{}或set()定义，通过元素本身访问。

3. 字符串切片语法为：s[start:end:step]，用于获取字符串中的一部分。其中start是起始索引（包含），end是结束索引（不包含），step是步长（默认值为1）。

4. 并集是两个集合中所有元素的集合；交集是两个集合中共同元素的集合；差集是第一个集合中独有的元素的集合。

### 四、编程题
1. 
```python
duplicate_list = [1, 2, 2, 3, 3, 4, 5, 5]
unique_list = list(set(duplicate_list))
print(unique_list)
```

2. 
```python
s = "hello world"
char_count = {}
for char in s:
    if char != " ":
        char_count[char] = char_count.get(char, 0) + 1
max_char = max(char_count, key=char_count.get)
print(max_char)
```

3. 
```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
union = set1 | set2
intersection = set1 & set2
difference1 = set1 - set2
difference2 = set2 - set1
print(f"并集：{union}")
print(f"交集：{intersection}")
print(f"差集（set1 - set2）：{difference1}")
print(f"差集（set2 - set1）：{difference2}")
```

4. 
```python
d = {"a": 1, "b": 2, "c": 3}
reversed_d = {v: k for k, v in d.items()}
print(reversed_d)
```
