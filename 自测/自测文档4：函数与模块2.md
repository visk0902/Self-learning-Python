## 三、简答题

1. Python函数参数类型及其特点：
   - 位置参数：按顺序传递的参数，必须按照定义顺序传递
   - 关键字参数：通过参数名传递的参数，可以不按顺序传递
   - 默认参数：在定义时指定默认值的参数，调用时可以不传递
   - 可变位置参数：使用*args接收任意数量的位置参数，返回元组
   - 可变关键字参数：使用**kwargs接收任意数量的关键字参数，返回字典

2. 递归函数的优缺点：
   优点：
   - 代码简洁，可读性高
   - 适合解决具有递归性质的问题，如斐波那契数列、阶乘等
   缺点：
   - 执行效率较低，因为需要多次调用函数
   - 占用较多内存，因为每次调用都会在栈中保存函数状态
   - 可能导致栈溢出，特别是当递归深度较大时

3. 装饰器的工作原理：
   - 装饰器是一个函数，它接收一个函数或类作为参数
   - 装饰器内部定义了一个包装函数，用于增强原函数的功能
   - 包装函数调用原函数，并在调用前后添加额外的代码
   - 装饰器返回包装函数，替换原函数
   - 使用@符号作为语法糖，可以简化装饰器的使用

4. Python模块和包的区别：
   - 模块是一个.py文件，包含Python代码
   - 包是一个包含__init__.py文件的目录，用于组织相关的模块
   - 模块用于封装函数、类和变量
   - 包用于组织和管理模块，形成层次结构
   - 导入模块使用import module或from module import...
   - 导入包中的模块使用from package import module或from package.module import...

## 四、实践操作题

1. 装饰器计算函数执行时间：
```python
import time

def calculate_time(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"函数{func.__name__}的执行时间为：{end_time - start_time:.4f}秒")
        return result
    return wrapper

@calculate_time
def test_func():
    time.sleep(1)
    return "测试完成"

test_func()
```

2. 接收任意数量参数并返回总和：
```python
def sum_all(*args, **kwargs):
    total = 0
    # 计算位置参数的总和
    for arg in args:
        if isinstance(arg, (int, float)):
            total += arg
    # 计算关键字参数的总和
    for value in kwargs.values():
        if isinstance(value, (int, float)):
            total += value
    return total

# 测试
print(sum_all(1, 2, 3, a=4, b=5))  # 输出：15
print(sum_all(1.5, 2.5, x=3.5, y=4.5))  # 输出：12.0
```
