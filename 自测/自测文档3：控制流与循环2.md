## 三、简答题

1. **if-elif-else语句的执行流程**：
   - 首先检查if条件，如果为真，执行对应的代码块，跳过elif和else
   - 如果if条件为假，检查第一个elif条件，如果为真，执行对应的代码块，跳过后面的elif和else
   - 依次检查每个elif条件，直到找到一个为真的条件，执行对应的代码块
   - 如果所有if和elif条件都为假，执行else对应的代码块

2. **for循环和while循环的区别**：
   - for循环用于遍历已知序列或可迭代对象，循环次数是确定的
   - while循环用于在条件为真时重复执行，循环次数是不确定的，需要根据条件退出
   - for循环的语法更简洁，适合遍历场景
   - while循环适合需要根据条件动态控制循环次数的场景

3. **break、continue和pass语句的区别**：
   - break语句用于终止整个循环，跳出循环体
   - continue语句用于跳过当前循环的剩余部分，继续下一次循环
   - pass语句用于占位，不执行任何操作，常用于语法需要但暂时不需要实现功能的场景

4. **range()函数的用法**：
   - range()函数用于生成一个整数序列，常用于for循环中
   - 语法：range(start, stop, step)
   - start：起始值，默认0
   - stop：结束值，不包含在序列中
   - step：步长，默认1
   - 示例：
     - range(5) → [0, 1, 2, 3, 4]
     - range(1, 5) → [1, 2, 3, 4]
     - range(1, 10, 2) → [1, 3, 5, 7, 9]
     - range(10, 0, -2) → [10, 8, 6, 4, 2]

## 四、编程题

1. **使用for循环打印1到100中的偶数**：

```python
# 方法1：使用for循环和if条件
print("1到100中的偶数：")
for i in range(1, 101):
    if i % 2 == 0:
        print(i, end=" ")
print()

# 方法2：使用range()函数的步长
print("1到100中的偶数：")
for i in range(2, 101, 2):
    print(i, end=" ")
print()
```

2. **使用while循环计算1到100的和**：

```python
# 使用while循环计算1到100的和
sum = 0
n = 1
while n <= 100:
    sum += n
    n += 1
print(f"1到100的和是：{sum}")
```
