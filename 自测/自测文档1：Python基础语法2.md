## 三、简答题

1. **Python的注释类型和用法**：
   - **单行注释**：使用#符号，用于注释一行代码
   - **多行注释**：使用三个单引号'''或三个双引号"""包裹，用于注释多行代码
   - **文档字符串**：使用三个单引号或三个双引号包裹，用于函数、类和模块的文档说明

2. **Python的缩进规则**：
   - Python使用缩进来表示代码块，而不是使用大括号
   - 缩进可以使用4个空格、2个空格或1个tab键，但必须保持一致
   - 同一代码块内的代码必须具有相同的缩进
   - 缩进的层数表示代码块的嵌套关系

3. **Python的字符串格式化方法**：
   - **%操作符**：使用格式说明符，如%s（字符串）、%d（整数）、%f（浮点数）等，用于格式化字符串
   - **format()方法**：使用{}作为占位符，通过位置、名称或索引来指定参数
   - **f-string**：使用f前缀和{}包裹变量，直接在字符串中插入变量值，Python 3.6+支持

4. **Python的变量命名规则**：
   - 变量名只能包含字母、数字和下划线
   - 变量名不能以数字开头
   - 变量名不能是Python关键字
   - 变量名区分大小写
   - 变量名应具有描述性，便于理解和维护

## 四、编程题

1. **计算两个数的和、差、积、商**：

```python
# 接收用户输入
num1 = float(input("请输入第一个数："))
num2 = float(input("请输入第二个数："))

# 计算结果
sum_result = num1 + num2
diff_result = num1 - num2
prod_result = num1 * num2

# 处理除数为零的情况
if num2 != 0:
    div_result = num1 / num2
    print(f"{num1} / {num2} = {div_result}")
else:
    print(f"{num1} / {num2} = 除数不能为零")

# 输出结果
print(f"{num1} + {num2} = {sum_result}")
print(f"{num1} - {num2} = {diff_result}")
print(f"{num1} * {num2} = {prod_result}")
```

2. **使用f-string格式化输出个人信息**：

```python
# 定义个人信息
name = "张三"
age = 25
gender = "男"
address = "北京市朝阳区"

# 使用f-string格式化输出
print(f"姓名：{name}")
print(f"年龄：{age}岁")
print(f"性别：{gender}")
print(f"地址：{address}")
print(f"\n个人信息：")
print(f"我叫{name}，今年{age}岁，{gender}，住在{address}。")

# 接收用户输入并输出
name = input("请输入您的姓名：")
age = input("请输入您的年龄：")
gender = input("请输入您的性别：")
address = input("请输入您的地址：")
print(f"\n您的个人信息：")
print(f"姓名：{name}")
print(f"年龄：{age}岁")
print(f"性别：{gender}")
print(f"地址：{address}")
print(f"\n自我介绍：")
print(f"大家好，我叫{name}，今年{age}岁，{gender}，来自{address}。")
```
