## 三、简答题

1. **类和对象的关系**：
   - 类是用于定义对象的模板，描述对象的属性和方法
   - 对象是类的实例，具有类定义的属性和方法
   - 一个类可以创建多个对象
   - 对象的属性和方法由其所属的类定义
   - 类是抽象的，对象是具体的

2. **封装、继承和多态的概念**：
   - **封装**：将数据和方法封装在类中，对外提供统一的接口，隐藏内部实现细节
   - **继承**：子类继承父类的属性和方法，实现代码复用。子类可以添加新的属性和方法，也可以重写父类的方法
   - **多态**：不同对象对同一消息作出不同的响应，即不同类的对象调用相同的方法时，会执行不同的实现

3. **实例方法、类方法和静态方法的区别**：
   - **实例方法**：第一个参数是self，表示当前对象，只能通过对象调用，访问实例属性和方法
   - **类方法**：第一个参数是cls，表示当前类，通过@classmethod装饰器定义，可以通过类或对象调用，访问类属性和方法
   - **静态方法**：不需要特殊的第一个参数，通过@staticmethod装饰器定义，可以通过类或对象调用，不能访问实例属性和方法，也不能访问类属性和方法（除非通过类名）

4. **Python中特殊方法的作用和常用的特殊方法**：
   - **作用**：特殊方法用于实现对象的特殊行为，如初始化、字符串表示、算术运算等
   - **常用的特殊方法**：
     - `__init__`：初始化方法，创建对象时调用
     - `__str__`：字符串表示方法，str(obj)或print(obj)时调用
     - `__repr__`：官方字符串表示方法，repr(obj)时调用
     - `__eq__`：相等比较方法，obj1 == obj2时调用
     - `__lt__`：小于比较方法，obj1 < obj2时调用
     - `__add__`：加法运算方法，obj1 + obj2时调用
     - `__len__`：长度方法，len(obj)时调用
     - `__getitem__`：获取元素方法，obj[key]时调用
     - `__setitem__`：设置元素方法，obj[key] = value时调用
     - `__delitem__`：删除元素方法，del obj[key]时调用

1. **学生类（Student）**：

```python
class Student:
    def __init__(self, name, age, gender, score):
        self._name = name
        self._age = age
        self._gender = gender
        self._score = score
    
    @property
    def name(self):
        """获取姓名"""
        return self._name
    
    @name.setter
    def name(self, value):
        """设置姓名"""
        if not isinstance(value, str):
            raise TypeError("姓名必须是字符串")
        self._name = value
    
    @property
    def age(self):
        """获取年龄"""
        return self._age
    
    @age.setter
    def age(self, value):
        """设置年龄"""
        if not isinstance(value, (int, float)):
            raise TypeError("年龄必须是数值类型")
        if value < 0 or value > 100:
            raise ValueError("年龄必须在0到100之间")
        self._age = value
    
    @property
    def gender(self):
        """获取性别"""
        return self._gender
    
    @gender.setter
    def gender(self, value):
        """设置性别"""
        if value not in ["男", "女"]:
            raise ValueError("性别必须是'男'或'女'")
        self._gender = value
    
    @property
    def score(self):
        """获取成绩"""
        return self._score
    
    @score.setter
    def score(self, value):
        """设置成绩"""
        if not isinstance(value, (int, float)):
            raise TypeError("成绩必须是数值类型")
        if value < 0 or value > 100:
            raise ValueError("成绩必须在0到100之间")
        self._score = value
    
    def display_info(self):
        """显示学生信息"""
        print(f"姓名：{self.name}")
        print(f"年龄：{self.age}")
        print(f"性别：{self.gender}")
        print(f"成绩：{self.score}")

# 测试学生类
```

2. **动物类（Animal）、狗类（Dog）和猫类（Cat）**：

```python
# 动物类
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def eat(self):
        print(f"{self.name}正在吃东西")
    
    def sleep(self):
        print(f"{self.name}正在睡觉")

# 狗类（继承自动物类）
class Dog(Animal):
    def bark(self):
        print(f"{self.name}正在汪汪叫")
    
    def eat(self):
        print(f"{self.name}正在吃狗粮")

# 猫类（继承自动物类）
class Cat(Animal):
    def meow(self):
        print(f"{self.name}正在喵喵叫")
    
    def eat(self):
        print(f"{self.name}正在吃猫粮")

# 函数，接收动物对象
def animal_action(animal):
    print(f"动物名称：{animal.name}")
    print(f"动物年龄：{animal.age}")
    animal.eat()
    animal.sleep()
    # 根据对象类型调用特殊方法
    if isinstance(animal, Dog):
        animal.bark()
    elif isinstance(animal, Cat):
        animal.meow()
    print()

# 创建对象
dog = Dog("旺财", 3)
cat = Cat("咪咪", 2)

# 调用函数
animal_action(dog)
animal_action(cat)
```
