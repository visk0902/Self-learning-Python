## 三、简答题

1. 面向对象编程的三大特性：
   - 封装：将数据和方法封装在一起，隐藏对象的内部实现细节，只对外提供接口
   - 继承：允许子类继承父类的属性和方法，实现代码复用
   - 多态：允许不同类的对象对同一方法做出不同的响应，提高代码的灵活性和可扩展性

2. 继承的优缺点：
   优点：
   - 实现代码复用
   - 建立类之间的层次关系
   - 提高代码的可维护性
   缺点：
   - 增加类之间的耦合度
   - 可能导致继承层次过深
   - 可能导致菱形继承问题

3. 多态的实现条件：
   - 继承：子类继承父类
   - 重写：子类重写父类的方法
   - 向上转型：将子类对象赋值给父类引用

4. 面向对象编程的设计原则：
   - 单一职责原则：一个类只负责一个功能
   - 开放封闭原则：对扩展开放，对修改封闭
   - 里氏替换原则：子类可以替换父类
   - 依赖倒置原则：依赖抽象，不依赖具体实现
   - 接口隔离原则：使用多个专门的接口，而不是一个统一的接口
   - 迪米特法则：一个对象应该对其他对象了解最少
   - 组合优于继承：优先使用组合，而不是继承

1. 动物类及其子类实现：
```python
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def eat(self):
        print(f"{self.name}正在吃东西")
    
    def sleep(self):
        print(f"{self.name}正在睡觉")

class Dog(Animal):
    def bark(self):
        print(f"{self.name}正在汪汪叫")
    
    def fetch(self):
        print(f"{self.name}正在追球")

class Cat(Animal):
    def meow(self):
        print(f"{self.name}正在喵喵叫")
    
    def climb(self):
        print(f"{self.name}正在爬树")

# 测试

dog.bark()     # 输出：旺财正在汪汪叫
dog.fetch()    # 输出：旺财正在追球

cat = Cat("咪咪", 2)
```

2. 银行账户类实现：
```python
class Account:
    def __init__(self, account_number, balance=0):
        self.account_number = account_number
        self.balance = balance
    
    def deposit(self, amount):
        if amount > 0:
            self.balance += amount
            print(f"存款成功！当前余额：{self.balance}")
        else:
            print("存款金额必须大于0！")
    
    def withdraw(self, amount):
        if amount > 0:
            if self.balance >= amount:
                self.balance -= amount
                print(f"取款成功！当前余额：{self.balance}")
            else:
                print(f"余额不足！当前余额：{self.balance}，取款金额：{amount}")
        else:
            print("取款金额必须大于0！")
    
    def check_balance(self):
        print(f"账号：{self.account_number}，当前余额：{self.balance}")

# 测试
account = Account("123456789", 1000)
account.check_balance()    # 输出：账号：123456789，当前余额：1000
account.deposit(500)       # 输出：存款成功！当前余额：1500
account.withdraw(2000)     # 输出：余额不足！当前余额：1500，取款金额：2000
account.withdraw(800)      # 输出：取款成功！当前余额：700
account.check_balance()    # 输出：账号：123456789，当前余额：700
```
