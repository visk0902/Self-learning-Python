## 三、简答题

1. **if-elif-else语句的执行流程**：
   - 首先检查if条件，如果为真，执行对应的代码块，跳过elif和else
   - 如果if条件为假，检查第一个elif条件，如果为真，执行对应的代码块，跳过后面的elif和else
   - 依次检查每个elif条件，直到找到一个为真的条件，执行对应的代码块
   - 如果所有if和elif条件都为假，执行else对应的代码块

2. **for循环和while循环的区别**：
   - for循环用于遍历已知序列或可迭代对象，循环次数是确定的
   - while循环用于在条件为真时重复执行，循环次数是不确定的，需要根据条件退出
   - for循环的语法更简洁，适合遍历场景
   - while循环适合需要根据条件动态控制循环次数的场景

3. **break语句和continue语句的区别**：
   - break语句用于终止整个循环，跳出循环体
   - continue语句用于跳过当前循环的剩余部分，继续下一次循环
   - break语句会结束循环，不再执行后续的循环迭代
   - continue语句只会跳过当前迭代，不会结束循环

4. **嵌套循环的使用场景**：
   - 遍历多维数据结构，如嵌套列表、矩阵等
   - 生成组合数据，如笛卡尔积
   - 实现复杂的算法，如排序算法、搜索算法等
   - 打印格式化输出，如九九乘法表、三角形等

## 四、编程题

1. **计算1到100的和**：

```python
# 方法1：使用for循环
sum = 0
for i in range(1, 101):
    sum += i
print(f"1到100的和是：{sum}")

# 方法2：使用while循环
sum = 0
i = 1
while i <= 100:
    sum += i
    i += 1
print(f"1到100的和是：{sum}")

# 方法3：使用等差数列公式
sum = (1 + 100) * 100 // 2
print(f"1到100的和是：{sum}")
```

2. **打印斐波那契数列的前10项**：

```python
# 方法1：使用for循环
n = 10
fib = [0, 1]  # 初始两项
for i in range(2, n):
    fib.append(fib[i-1] + fib[i-2])
print("斐波那契数列的前10项：")
for num in fib:
    print(num, end=" ")
print()

# 方法2：使用while循环
n = 10
fib = [0, 1]
i = 2
while len(fib) < n:
    fib.append(fib[i-1] + fib[i-2])
    i += 1
print("斐波那契数列的前10项：")
for num in fib:
    print(num, end=" ")
print()

# 方法3：不使用列表，直接打印
a, b = 0, 1
print("斐波那契数列的前10项：")
print(a, end=" ")
print(b, end=" ")
for _ in range(8):  # 已经打印了2项，还需要打印8项
    c = a + b
    print(c, end=" ")
    a, b = b, c
print()
```
