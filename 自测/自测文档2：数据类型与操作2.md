## 三、简答题

1. **Python的注释类型和用法**：
   - **单行注释**：使用#符号，用于注释一行代码
   - **多行注释**：使用三个单引号'''或三个双引号"""包裹，用于注释多行代码
   - **文档字符串**：使用三个单引号或三个双引号包裹，用于函数、类和模块的文档说明

2. **Python的缩进规则**：
   - Python使用缩进来表示代码块，而不是使用大括号
   - 缩进可以使用tab键或空格，但必须保持一致
   - 通常使用4个空格作为缩进
   - 缩进的层数表示代码块的嵌套关系
   - 同一代码块内的代码必须具有相同的缩进

3. **Python的字符串格式化方法**：
   - **%操作符**：使用格式说明符，如%s、%d、%f等，用于格式化字符串
   - **format()方法**：使用{}作为占位符，通过位置、名称或索引来指定参数
   - **f-string**：使用f前缀和{}包裹变量，直接在字符串中插入变量值

4. **Python的运算符类型和常见运算符**：
   - **算术运算符**：+、-、*、/、//、%、**
   - **比较运算符**：==、!=、>、<、>=、<=
   - **逻辑运算符**：and、or、not
   - **身份运算符**：is、is not
   - **成员运算符**：in、not in
   - **赋值运算符**：=、+=、-=、*=、/=、//=、%=、**=

## 四、编程题

1. **接收用户输入并格式化输出**：

```python
# 接收用户输入
name = input("请输入您的姓名：")
age = input("请输入您的年龄：")
gender = input("请输入您的性别：")

# 使用f-string格式化输出
print(f"姓名：{name}")
print(f"年龄：{age}")
print(f"性别：{gender}")
print(f"您的姓名是{name}，年龄是{age}岁，性别是{gender}。")
```

2. **计算两个数的运算结果并格式化输出**：

```python
# 定义两个数
a = 10
b = 3

# 计算结果
sum_result = a + b
diff_result = a - b
prod_result = a * b
div_result = a / b
floor_div_result = a // b
mod_result = a % b

# 使用format()方法格式化输出
print("{} + {} = {}".format(a, b, sum_result))
print("{} - {} = {}".format(a, b, diff_result))
print("{} * {} = {}".format(a, b, prod_result))
print("{} / {} = {}".format(a, b, div_result))
print("{} // {} = {}".format(a, b, floor_div_result))
print("{} % {} = {}".format(a, b, mod_result))

# 使用format()方法的其他用法
print("{0} + {1} = {2:.2f}".format(a, b, sum_result))  # 保留两位小数
print("{a} + {b} = {sum}".format(a=a, b=b, sum=sum_result))  # 使用名称
print("{:>5} + {:>5} = {:>5}".format(a, b, sum_result))  # 右对齐，宽度为5
```
